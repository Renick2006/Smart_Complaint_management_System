# Smart_Complaint_management_System
## Description:
Smart Complaint Management System is an ML-powered web application that automatically classifies customer complaints into categories (Billing, Delivery, Product) and predicts urgency levels. Complaints are stored in a database and managed through an interactive admin dashboard featuring real-time analytics, pie charts, and resolution tracking.
## Features:
Features:

Submit complaints with email

ML-based category + urgency prediction

Admin dashboard with interactive pie charts

Complaint acknowledgement tracking
## Tech Stack:
Python, Flask, Scikit-learn, TF-IDF, SQLite, Chart.js, Bootstrap
